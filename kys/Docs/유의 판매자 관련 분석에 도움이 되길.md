# 유의 판매자 관련 분석

## 1. 데이터 구조 개요

데이터 간의 유기적인 연결 관계는 다음과 같습니다.

```mermaid
graph LR
    A[배송(Orders)] --- B[상품(Items)]
    B --- C[리뷰(Reviews)]
    B --- D[물건 상세(Products)]

```

> **구조 요약**: 배송 - 상품 - 리뷰 (상품은 물건 상세 정보와 연결)

---

## 2. 테이블 상세 정의

### Orders (배송)

- **데이터 상태**: 결측치 있음 (Null O)

| 컬럼명                          | 설명                              | 비고 |
| ------------------------------- | --------------------------------- | ---- |
| `order_id`                      | 주문 고유 ID                      |      |
| `customer_id`                   | 고객 고유 ID                      |      |
| `order_status`                  | 주문 상태                         |      |
| `order_purchase_timestamp`      | 주문 구매 타임스탬프              |      |
| `order_approved_at`             | 지불 승인 타임스탬프              |      |
| `order_delivered_carrier_date`  | 상품이 물류사에 전달된 타임스탬프 |      |
| `order_delivered_customer_date` | 실제 배송된 날짜                  |      |
| `order_estimated_delivery_date` | 고객에게 통보된 예상 배송일       |      |

### Items (상품)

- **데이터 상태**: **결측치 없음 (Null X)**

| 컬럼명                | 설명              | 비고                               |
| --------------------- | ----------------- | ---------------------------------- |
| `order_id`            | 주문 고유 ID      |                                    |
| `order_item_id`       | 주문 내 상품 순번 |                                    |
| `product_id`          | 상품 고유 ID      |                                    |
| `seller_id`           | 판매자 고유 ID    |                                    |
| `price`               | 상품 가격         |                                    |
| `shipping_limit_date` | 판매자 발송 기한  | 판매자가 주문을 발송해야 하는 기한 |
| `freight_value`       | 배송비            |                                    |

### Review (리뷰)

- **데이터 상태**: 결측치 있음 (Null O)

| 컬럼명                   | 설명         | 비고 |
| ------------------------ | ------------ | ---- |
| `review_id`              | 리뷰 고유 ID |      |
| `order_id`               | 주문 고유 ID |      |
| `review_score`           | 리뷰 평점    |      |
| `review_comment_title`   | 리뷰 제목    |      |
| `review_comment_message` | 리뷰 내용    |      |

### Products (물건)

- **데이터 상태**: 결측치 있음 (Null O)

| 컬럼명                        | 설명                  | 비고     |
| ----------------------------- | --------------------- | -------- |
| `product_id`                  | 상품 고유 ID          |          |
| `product_category_name`       | 카테고리명            |          |
| `product_name_lenght`         | 상품명 길이           |          |
| `product_description_lenght`  | 상품 설명 길이        |          |
| `product_photos_qty`          | 상품 사진 수          |          |
| `product_weight`              | 무게                  |          |
| `product_length/height/width` | 규격 (길이/높이/너비) | 단위: cm |

---

## 3. 인사이트 도출을 위한 주요 컬럼

### 기본 정보 (PK / 가격 / 배송비 / 주문상태)

- `order_id`
- `price`
- `freight_value`
- `order_status`

### ℹ제품 정보 (카테고리 / 설명 / 사진)

- `product_category_name`
- `product_name_lenght`
- `product_description_lenght`
- `product_photos_qty`

### 날짜 및 시간 (Lead Time 분석용)

- **판매자 책임**: `shipping_limit_date` (발송 기한)
- **주문 시작**: `order_purchase_timestamp` (구매)
- **결제**: `order_approved_at` (승인)
- **물류 이동**: `order_delivered_carrier_date` (물류사 전달)
- **배송 완료**: `order_delivered_customer_date` (도착)
- **예상일**: `order_estimated_delivery_date` (도착 예정)

### 리뷰 (평판 분석)

- `review_score`
- `review_comment_title`
- `review_comment_message`
