# Olist 브라질 이커머스 데이터 분석 프로젝트

## 프로젝트 개요

본 프로젝트는 **2016-2018년 브라질 Olist 플랫폼의 실제 전자상거래 주문 데이터**를 활용하여 이커머스 비즈니스 인사이트를 도출하고 머신러닝 모델을 구축하는 것을 목표로 합니다.

### 핵심 가치

- **데이터 규모**: 약 100,000건의 실제 주문 데이터
- **분석 범위**: 주문, 결제, 배송, 리뷰, 고객, 판매자, 제품 등 전 과정 분석
- **비즈니스 목적**: 서비스 개선점 파악 및 마케팅 전략 수립

---

## 주요 분석 목표

### 1. 분류: 리뷰 만족도 예측 (Classification)

- **목표**: 고객 리뷰 점수를 기반으로 긍정/부정 리뷰 예측
- **핵심 과제**:
  - 불만족 리뷰의 주요 원인 파악 (배송 지연, 제품 카테고리, 결제 방식 등)
  - 만족도에 가장 큰 영향을 미치는 독립 변수 식별

### 2. 회귀: 배송 소요 일수 예측 (Regression)

- **목표**: 구매 시점부터 실제 배송 완료까지의 소요 기간 예측
- **핵심 과제**:
  - 주문 지역과 판매자 거점 간의 거리, 배송사 정보 등을 변수로 활용
  - 예측값과 실제값 비교를 통한 물류 계획 최적화 방안 제안

### 3. 군집: 고객 세그먼트 도출 (Clustering)

- **목표**: 유사한 구매 성향을 가진 고객 그룹 정의 및 프로파일링
- **핵심 과제**:
  - 구매 횟수, 평균 구매액(Monetary), 결제 유형, 지역 정보를 활용한 군집화
  - 군집별 특성에 따른 마케팅 타겟팅 전략(RFM 분석 등) 수립

데이터는 관계형 데이터베이스 형태로 **9개의 주요 테이블**로 구성됩니다:

| 테이블명                            | 설명           | 주요 컬럼                                                         |

| `olist_order_items_dataset`         | 주문 항목 상세 | 상품 ID, 판매자 ID, 가격, 배송비                                  |
| `olist_customers_dataset`           | 고객 정보      | 고객 ID, 우편번호, 도시, 주                                       |
| `olist_products_dataset`            | 제품 정보      | 카테고리, 상품명 길이, 사진 수, 무게, 치수                        |
| `olist_sellers_dataset`             | 판매자 정보    | 판매자 ID, 위치                                                   |
| `olist_order_payments_dataset`      | 결제 정보      | 결제 유형, 할부 횟수, 금액                                        |
| `olist_order_reviews_dataset`       | 리뷰 정보      | 리뷰 점수(1-5점), 코멘트                                          |
| `olist_geolocation_dataset`         | 지리 정보      | 위도, 경도                                                        |
| `product_category_name_translation` | 카테고리 번역  | 포르투갈어→영어 번역                                              |

## 데이터 품질 체크리스트

분석 전반에 걸쳐 다음 체크리스트를 검증해야 합니다:

1. **Cluster Mapping Integrity**: 각 고객이 단일 클러스터에만 매핑되는지 확인 (일대일 매핑)
2. **Missing Data Check**: 데이터셋 내 결측치(Null/NaN) 비율 확인
3. **Zero-Inflation Check**: 결측치가 아니더라도 0 값의 빈도가 높은지 확인 (희소 데이터)
4. **Distribution & Correlation**:
   - 데이터의 전체 분포 확인
   - 피처 간 상관계수 확인 (다중공선성 체크)
5. **Data Irregularity**: 데이터 패턴의 불규칙성 또는 노이즈 확인
6. **Feature Semantics & Seasonality**:
   - 각 컬럼의 개념적 의미 파악
   - 월말/월초, 연말/연초 등 특정 시기 제외 고려
7. **Binary Features**: 컬럼 값이 이진(0/1 또는 True/False)인지 확인
8. **Cluster Balance**: 결과 클러스터의 비율이 과도하게 치우쳐 있지 않은지 확인
9. **Evaluation Metrics**:
   - `confusion_matrix` (필수)
   - `classification_report` (필수)

## 브라질 이커머스 시장 컨텍스트 (2016-2018)

### 시장 규모 및 성장률

- **2016년**: BRL 44.4억 (USD 13-14억), 성장률 7%
- **2017년**: BRL 47.7억 (USD 14.4-18.9억), 성장률 8%
- **2018년**: BRL 53.5-77.5억 (USD 15-21.3억), 성장률 12-21%

**특징**: 2016-2017년은 브라질 경제 위기로 인해 성장률이 둔화되었으나, 2018년부터 회복 국면 진입

### 결제 수단 분포

- **신용카드**: 66-78% (압도적 1위)
  - 할부 결제 문화: 약 60%가 3-12개월 할부 선택
- **Boleto Bancário**: 15-24% (2위)
  - 은행 계좌 미보유 고객의 주요 결제 수단
- **은행 이체**: 약 7%
- **기타**: Cash on Delivery 5%, 기타 1%

### 주요 마켓플레이스

- **Mercado Livre**: 브라질 시장 점유율 약 27% (압도적 1위)
- **Magazine Luiza**: 전자제품·가구 중심
- **Amazon Brazil**: 2016-2018년은 초기 단계

### 소비자 행동 특성

- **배송 시간 민감도 높음**: 크로스보더 구매 포기 이유 1위 (44%)
- **가격 비교 활발**: 온라인에서 가격 비교하지만 실제 구매 전환율은 낮음
- **할부 문화**: 소액 구매에도 할부 사용, 구매력 확대 수단

## 판매자 페르소나

### 1️Fresh Starter (새내기 소상공인)

- 가입 후 0-6개월, 월 10-50건 주문
- 월 매출 BRL 5,000-50,000
- **효율성 지표**: 배송 지연률 < 20%, 평균 리뷰 ≥ 3.5점, 재구매율 > 20%

### Growing SMB (성장하는 중소 판매자)

- 가입 후 6-24개월, 월 100-500건 주문
- 월 매출 BRL 50,000-500,000
- **효율성 지표**: 배송 지연률 < 15%, 평균 리뷰 ≥ 4.2점, 월 주문 증가율 > 10%

### Stable Established (안정적 중견 판매자)

- 가입 후 24개월 이상, 월 500-2,000건 이상 주문
- 월 매출 BRL 500,000-2,000,000 이상
- **효율성 지표**: 배송 지연률 < 10%, 평균 리뷰 ≥ 4.5점, 이윤율 > 20%, 재구매율 > 40%

### Niche Specialist (틈새 전문가)

- 가입 후 1-3년 이상, 월 50-300건 주문
- 월 매출 BRL 100,000-300,000
- **효율성 지표**: 평균 리뷰 ≥ 4.3점, 재구매율 > 35%, 반품/환불 비율 < 3%

### At-Risk / Bad Actors (위험 판매자)

- 운영 불안정, 저평점 비율 > 15%, 배송 지연률 > 30%
- **위험 신호**: 클레임/환불 비율 > 10%, 배송비 폭리, 부정적 코멘트
