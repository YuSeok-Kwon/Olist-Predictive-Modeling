## 1. 배송 분석

### 1.1 배송 시간 정의

| 지표                 | 계산 방식                                                       | 책임 주체 |
| -------------------- | --------------------------------------------------------------- | --------- |
| **판매자 처리 시간** | `order_delivered_carrier_date - order_approved_at`              | 판매자    |
| **물류 배송 시간**   | `order_delivered_customer_date - order_delivered_carrier_date`  | 물류사    |
| **전체 배송 시간**   | `order_delivered_customer_date - order_purchase_timestamp`      | (종합)    |
| **배송 지연**        | `order_delivered_customer_date - order_estimated_delivery_date` | -         |

### 1.2 위치 기반 분석 (동일 주 vs 다른 주)

| 지표                 | 동일 주            | 다른 주              | 차이                  |
| -------------------- | ------------------ | -------------------- | --------------------- |
| **판매자 처리 시간** | 2.3일 (중앙값 1일) | 2.4일 (중앙값 1일)   | **0.1일** (거의 없음) |
| **물류 배송 시간**   | 4.3일 (중앙값 3일) | 11.3일 (중앙값 9일)  | **7.0일**             |
| **전체 배송 시간**   | 7.5일 (중앙값 6일) | 14.6일 (중앙값 12일) | **7.1일**             |

- 동일 주 배송 비율: **36.2%**
- 지역 간 배송 시간 차이의 **70% 이상이 물류 배송 시간**에서 발생
- 판매자 처리 시간은 지역과 **무관** (약 2일 일정)

## 6. 피처 중요도 요약

### 6.1 리뷰 점수 예측에 중요한 피처

| 순위 | 피처               | 영향도     | 비고                       |
| ---- | ------------------ | ---------- | -------------------------- |
| 1    | **배송 지연 여부** | ⭐⭐⭐⭐⭐ | 가장 강력한 예측 변수      |
| 2    | **동일 주 배송**   | ⭐⭐⭐⭐   | 물류 배송 시간에 직접 영향 |
| 3    | **물류 배송 시간** | ⭐⭐⭐⭐   | 지연과 높은 상관           |
| 4    | **주문 상품 수**   | ⭐⭐⭐     | 복잡한 주문 = 낮은 점수    |
| 5    | **특수 기간**      | ⭐⭐⭐     | 블랙프라이데이 등          |
| 6    | **카테고리**       | ⭐⭐       | 일부 카테고리 영향         |
| 7    | **제품 정보 유무** | ⭐⭐       | 있으면 약간 좋음           |
| 8    | **주문 금액**      | ⭐         | 약한 영향                  |
| 9    | **요일/시간대**    | ⭐         | 거의 무관                  |

### 6.2 피처 엔지니어링 권장

| 피처                     | 생성 방법                       | 활용             |
| ------------------------ | ------------------------------- | ---------------- |
| `same_state`             | customer_state == seller_state  | 이진 피처        |
| `processing_days`        | carrier_date - approved_at      | 판매자 성과 지표 |
| `shipping_days`          | delivered_date - carrier_date   | 물류 성과 지표   |
| `delivery_delay`         | delivered_date - estimated_date | 핵심 예측 변수   |
| `seller_processing_diff` | 판매자 처리 시간 - 주 평균      | 문제 판매자 식별 |
| `is_holiday_season`      | 블랙프라이데이/연말 여부        | 이진 피처        |
| `has_product_info`       | 사진/설명 존재 여부             | 이진 피처        |

## 7. 결론

### 7.1 핵심 발견

| 분석 영역     | 핵심 발견                                         |
| ------------- | ------------------------------------------------- |
| **배송**      | 지연 시 리뷰 1.6점 하락, 동일 주 배송 시 7일 빠름 |
| **제품 정보** | "있느냐 없느냐"만 중요, 정보량은 무관             |
| **판매 구조** | 상품 수 많을수록 리뷰 하락, 금액은 약한 영향      |
| **시간**      | 블랙프라이데이만 유의미, 요일/시간대 무관         |
| **지역**      | 북동부 지역 물류 병목                             |

### 7.2 비즈니스 제언

| 영역            | 제언                                |
| --------------- | ----------------------------------- |
| **물류**        | 북동부 지역 물류 인프라 강화        |
| **판매자 관리** | 처리 시간 느린 판매자 모니터링/교육 |
| **카테고리**    | 대형 제품 별도 배송 프로세스 수립   |
| **시즌 대응**   | 블랙프라이데이 사전 물량 확보       |
| **제품 정보**   | 기본 정보(사진 1장, 설명) 필수화    |
